## About

Photo library for organizing and editing your images.

## Features

- Photo organization
- Albums
- Editing tools
- Memories
- Search

## Installation

### From App Store
Open the **App Store** in zOS and search for "Photos".

### Via npm
```bash
npm install @zos-apps/photos
```

## Usage

- Import photos
- Create albums
- Edit with tools

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `E` | Edit |
| `I` | Info |
| `Cmd+A` | Select all |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
